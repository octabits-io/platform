# Proprietary Packages

Packages in this directory are licensed under the Business Source License 1.1 (BSL).

Each package must include its own `LICENSE` file with the full BSL text.

You can find the BSL 1.1 template at: https://mariadb.com/bsl11/

When adding a new package here, copy the template and fill in:
- **Licensor**: The Octabits Authors
- **Licensed Work**: (package name and version)
- **Change Date**: 4 years from release date
- **Change License**: MIT
- **Additional Use Grant**: (define what free usage is allowed, e.g., non-production use)
